# iptv features

1. Automatic Subtitle discovery, 
2. adding m3u list and xstream separately
3. Add, delete and edit many profiles
4. When adding a profile, do not capitalize the profile name,
5. Connection speed when watching videos
6. Change resolution, vision
7. Changing audio file visibility
8. Subtitle loading, selection, auto-selection
9. connection Status indicator
10. Membership end date indicator
11. Live, Movie and series indicator
12. Option to search live, movies and TV series separately
13. live, search movie and series categories, select by category
14. Add and remove favorites, see favorites on the home screen
15. Show instant connection speed while watching video
16. Displaying movie and TV series posters, plot, etc.
17. IMDB score imaging
18. Option to skip intro while watching a series
19. Skip to next episode option when watching a series
20. Option to select default audio and subtitle language on the Home Screen
21. Show Upcoming Matches on the main screen
22. Not showing adult content on the home screen
23. Vertical and horizontal viewing option
24. Automatically switch to a profile when it is added
25. `ConnectionPool` keeps the connection to the server warm and zapping (channel switching) is much faster.
26. Using state-of-the-art Exoplayer
27.  Multiple Playlist Support
28. Automatically switch to a newly added profile when a profile is added
